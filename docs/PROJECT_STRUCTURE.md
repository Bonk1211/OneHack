## ğŸ¯ Key Directories

### `src/` - Source Code
- **components/**: React UI components
- **hooks/**: Custom React hooks for state management
- **utils/**: Helper functions and utilities
- **styles/**: CSS stylesheets (one per component)
- **config/**: Configuration files
- **data/**: Static game data (items, cats, etc.)

### `public/` - Static Assets
- **Animation/**: Cat sprite animations
- **images/**: Game images (items, UI elements)
- **audio/**: Sound effects
- **Music/**: Background music
- **UI/**: UI component assets

### `packages/` - Smart Contracts
- **pomodoro_game/**: Move contracts for OneChain
- **contracts/**: Legacy EVM contracts (if any)

### `docs/` - Documentation
- User guides
- Architecture documentation
- Project structure

### `scripts/` - Utility Scripts
- Treasury funding scripts
- Validation scripts
- Deployment helpers

## ğŸ“ File Naming Conventions

- **Components**: PascalCase (e.g., `PomodoroTimer.jsx`)
- **Utilities**: camelCase (e.g., `gameState.js`)
- **Styles**: Component name + `.css` (e.g., `PomodoroTimer.css`)
- **Config**: lowercase (e.g., `onechain.js`)
